# Machine Learning Specialization – Practical Exercises

Practical exercises (labs) and solutions for the **Machine Learning Specialization**, offered by **DeepLearning.AI** and **Stanford Online** on Coursera, taught by **Andrew Ng**.

## 📚 About the Specialization

The specialization consists of 3 main courses:

1. **Supervised Machine Learning: Regression and Classification**
2. **Advanced Learning Algorithms**
3. **Unsupervised Learning, Recommenders, Reinforcement Learning**

## 📂 Repository Contents

| Folder | Content |
|---|---|
| `C1_W1` | Course 1 – Week 1 |
| `C1_W2` | Course 1 – Week 2 |
| `C1_W3` | Course 1 – Week 3 |
| `C2_W1` | Course 2 – Week 1 |
| `C2_W2` | Course 2 – Week 2 |
| `C2_W3` | Course 2 – Week 3 |
| `C2_W4` | Course 2 – Week 4 |
| `C3_W2` | Course 3 – Week 2 |
| `C3_W3` | Course 3 – Week 3 |

Each folder contains the Jupyter Notebooks for that week's exercises and solutions.

## 🛠️ Tools Used

- Python
- Jupyter Notebook
- NumPy, Pandas, Matplotlib
- scikit-learn / TensorFlow (depending on the exercise)

## 🚀 How to Run

```bash
git clone https://github.com/fares-ahmed7/Machine-Learning-Specialization-Practical-Exercises.git
cd Machine-Learning-Specialization-Practical-Exercises
pip install -r requirements.txt 
jupyter notebook
```
